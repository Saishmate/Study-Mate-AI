# StudyMate AI

StudyMate AI is an AI-powered study assistant that helps students learn faster from their notes.

## Features

* AI Notes Summarization
* Important Question Generation
* Flashcard Generation
* MCQ Quiz Generation
* User Authentication
* Note Management
* Gemini AI Integration

## Tech Stack

* React
* Node.js
* PostgreSQL
* Gemini API
* Replit

## Future Enhancements

* PDF Upload
* Download Summary as PDF
* Study Progress Tracking
* Subject-wise Analytics
